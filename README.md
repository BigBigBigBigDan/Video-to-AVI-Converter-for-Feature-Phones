
# Video to AVI Converter for Feature Phones

Скрипт для Termux, конвертирующий видео в формат AVI, оптимизированный для кнопочных телефонов (MXMID G88 Pro и аналогичных).

## Возможности

- Конвертация одиночных файлов
- Пакетная обработка целых папок (включая подпапки)
- Автоматический пропуск уже сконвертированных файлов
- Прогресс обработки и итоговая статистика

## Требования

- Termux (Android)
- ffmpeg

```bash
pkg update && pkg upgrade
pkg install ffmpeg
```

Установка

```bash
nano ~/convert_to_avi.sh
# Вставить код скрипта
chmod +x ~/convert_to_avi.sh
```

Использование

Одиночный файл

```bash
~/convert_to_avi.sh video.mp4
```

Папка с видео

```bash
~/convert_to_avi.sh /storage/emulated/0/Download
```

Текущая папка

```bash
~/convert_to_avi.sh .
```

Параметры кодирования

Параметр Значение
Контейнер AVI
Видеокодек MJPEG
Качество видео -q:v 5
Разрешение 352x288 (CIF)
Частота кадров 24 fps
Аудиокодек MP3 (libmp3lame)
Битрейт аудио 192 Кбит/с
Частота дискретизации 48 кГц
Каналы 2 (стерео)

Поддерживаемые входные форматы

mp4, mkv, avi, mov, flv, wmv, webm, 3gp, m4v, ts

Результат

Сконвертированные файлы сохраняются в ту же папку с суффиксом _fixed.avi.

Настройка качества

Качество видео регулируется параметром -q:v (шкала 1–31, где 1 — лучшее):

```bash
-q:v 3   # высокое качество
-q:v 5   # сбалансированное (по умолчанию)
-q:v 8   # меньший размер файла
```
